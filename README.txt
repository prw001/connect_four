How to play:
The game requires 2 players.  Each player takes turn adding a colored token - red or blue to one of the columns on the board.  The first player to make four-in-a-row wins.

Features:
Colorized, allows for saving and loading of games, or pick up a new game anytime.

-To run the game:

Requires Ruby version 2.5.0 or later

Download this repository.  In terminal, change directories to the ‘lib’ file of this repository, and run program by entering ‘ruby connect_four.rb’.


####Before running the test suite:#####

GameBoard.rb and connect_four.rb ‘require’ lines must be modified - see the comments at the top line of each file.

Comment out the menu() call at the bottom of connect_four.rb
